# 人物设计参考

## 人物欲望双层设计

每个主角必须同时拥有表层欲望和深层需求。

### 表层欲望（主角以为自己想要的）

主角自己意识到的、可以用行动去追求的目标：

- 我要赢
- 我要逃走
- 我要找到某个人
- 我要证明自己
- 我要隐藏秘密
- 我要完成任务
- 我要报仇
- 我要保护某人
- 我要获得自由
- 我要赚到钱

### 深层需求（主角真正缺失的）

主角没有意识到的、真正需要被填补的情感缺口：

- 被爱
- 被理解
- 被认可
- 学会放手
- 面对愧疚
- 接受真实的自己
- 原谅自己
- 信任他人
- 承认脆弱
- 与过去和解

### 设计原则

表层欲望驱动剧情——主角因为它而行动。

深层需求驱动人物弧光——故事的真正主题藏在这里。

最好的故事是：主角追求表层欲望的过程中，被迫面对深层需求。

## 人物缺陷类型库

完美人物不成立。以下是常见的人物缺陷：

| 缺陷 | 表现 | 适合的故事类型 |
|------|------|--------------|
| 懦弱 | 关键时刻退缩、回避冲突 | 成长型、英雄旅程 |
| 偏执 | 固执己见、不听劝告 | 悲剧、职场、家庭 |
| 自卑 | 不相信自己值得 | 爱情、青春、逆袭 |
| 控制欲 | 必须掌控一切、不允许意外 | 家庭、职场、亲子 |
| 逃避 | 不愿面对真相或责任 | 悬疑、家庭、成长 |
| 嘴硬心软 | 用攻击性保护脆弱 | 喜剧、爱情、亲情 |
| 不会表达爱 | 关心人的方式让人误解 | 亲情、爱情、温情 |
| 害怕被抛弃 | 先推开别人以免被抛弃 | 爱情、友情、创伤 |
| 想赢但怕输 | 不敢全力以赴因为怕失败 | 竞技、创业、成长 |
| 想爱但不敢靠近 | 渴望亲密但一旦靠近就退缩 | 爱情、治愈 |

## 人物弧光模式

人物弧光是人物从故事开头到结尾发生的内在变化。

### 正向弧光（成长型）

人物从缺陷走向成长：

- 从逃避到面对
- 从冷漠到承担
- 从控制到放手
- 从虚荣到诚实
- 从讨好别人到接受自己
- 从不相信爱到愿意相信
- 从被规则驯化到主动选择自由

### 负向弧光（堕落型）

人物从正常走向崩坏：

- 从善良到冷酷
- 从理想到妥协
- 从信任到猜疑
- 从清醒到沉迷

### 平弧光（坚守型）

人物不变，但改变了周围的世界：

- 世界要他放弃，他没有
- 所有人妥协了，他没有
- 最终证明他的坚持是对的（或付出了代价证明他的坚持值得尊敬）

## 潜台词写法

### 原则

> 台词表面说一件事，底下藏另一件事。

角色不会直接说出自己最真实的情感。他们会用遮掩、转移、反话、象征来表达。

### 低级台词 vs 高级台词

| 低级（直说） | 高级（潜台词） | 底下藏的是 |
|-------------|-------------|-----------|
| 我很想你 | 你家楼下那家面馆，还开着吗？ | 我一直在想你，我想回到过去 |
| 我很害怕 | 今天风好大啊 | 我想找个借口不出门 |
| 我爱你 | 你衣服扣子扣错了 | 我一直在看你，关心你的每个细节 |
| 我很抱歉 | 你知道那条路修好了吗？ | 我还记得那次事故，一直无法释怀 |
| 我不想失去你 | 这个菜你多吃点 | 我不知道怎么说，只能用行动表达 |

### 写台词时的四个自问

1. 角色真正想说什么？
2. 他为什么不能直接说？
3. 他用什么话遮掩真实情绪？
4. 观众能不能听出没说出口的东西？

### 常见遮掩方式

- **说反话**："走吧，我不需要你。"（其实最需要）
- **转移话题**："算了，不说了，你吃饭了没？"（不想面对刚才的话题）
- **用动作代替**：不说"原谅"，而是默默把对方的杯子倒满水
- **用象征代替**：不说"我还爱你"，而是把他离开时忘在这里的伞修好了放在门口
- **用幽默遮盖**：用玩笑化解最痛的话题

## 五类人物立场系统

每个剧本必须同时设计以下五类人物。每类人物都不允许是纯好人或纯坏人。

| 类型 | 定义 | 核心要求 |
|------|------|----------|
| **主角** | 想解决问题，但有软肋 | 必须有恐惧、私心、亏欠或秘密；不是完美的人 |
| **对手** | 阻止主角，但不是纯坏人 | 必须有自己的理由；他的理由在某些角度上必须成立 |
| **亲密关系者** | 与主角有情感牵连 | 既可能帮助主角，也可能伤害主角 |
| **旁观者** | 看似无关，其实掌握关键秘密 | 沉默必须有代价；他不说出来是因为有自己的顾虑 |
| **牺牲者** | 死者、失踪者、被误解的人、被牺牲的小人物 | 代表情感锚点里最痛的部分；是真相的载体 |

### 每个人物必须具备五个维度

| 维度 | 问题 |
|------|------|
| **欲望** | 他/她最想要什么？ |
| **恐惧** | 他/她最害怕什么？ |
| **秘密** | 他/她隐瞒了什么？ |
| **不得不做的事** | 他/她为什么没有选择？ |
| **不愿承认的真相** | 他/她内心知道但不敢面对的是什么？ |

### 设计原则

- 对手的理由必须能成立——观众能理解他为什么这么做，即使不认同
- 旁观者的沉默必须有代价——他说出来会失去什么，这才让沉默合理
- 牺牲者不能只是被动的受害者——他/她的存在和选择必须推动真相的揭开
- 亲密关系者的"伤害"必须出于爱或恐惧，而非恶意

### 三组强人物关系钩子

每部剧本至少设计三组强人物关系，每组必须回答四个问题：

1. 他们表面是什么关系？
2. 他们真正欠对方什么？
3. 他们最不能说出口的话是什么？
4. 他们最后会因为真相变成什么关系？

常见强关系类型：父母与孩子、夫妻、前任、兄弟姐妹、老板与员工、施害者与受害者、背叛者与被背叛者、合伙人与出卖者。
