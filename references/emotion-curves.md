# 五种情绪曲线模板

## 1. 情感型

适用于：家庭片、亲情片、爱情片、人物传记、温情广告

```
好奇 → 心疼 → 压抑 → 爆发 → 释怀 → 余味
```

**节奏要点**：
- 开头用一个微妙的异常引发好奇
- 中段逐步揭示人物的痛苦或牺牲
- 情绪压抑到极点后，用一个小动作或小细节完成情感爆发
- 结尾收在克制中，让观众自己补完情绪

**名场面常出现在**：爆发到释怀的转折处

---

## 2. 悬念型

适用于：悬疑片、推理片、反转短片、社会议题片

```
异常 → 紧张 → 误导 → 发现真相 → 反转 → 沉默
```

**节奏要点**：
- 开头设置一个无法解释的异常
- 中段提供看似合理但实际错误的解释（误导）
- 真相的揭示必须让观众重新理解前面所有情节
- 反转后留一拍沉默，让信息量自行冲击观众

**名场面常出现在**：反转揭示的瞬间

---

## 3. 喜剧型

适用于：喜剧短片、搞笑广告、荒诞剧、讽刺片

```
荒诞 → 尴尬 → 失控 → 反差 → 温暖 → 回味
```

**节奏要点**：
- 开头建立一个荒诞的前提或规则
- 中段让局面逐步失控，制造尴尬和社死感
- 在最失控的时刻突然出现反差（硬汉哭了、冷面人笑了）
- 用一个温暖的小细节收尾，让笑中带泪

**名场面常出现在**：失控到反差的转折处

---

## 4. 爽感型

适用于：逆袭短片、短剧、职场片、竞技片、复仇片

```
受压 → 忍耐 → 反击 → 逆转 → 爽点 → 价值升华
```

**节奏要点**：
- 开头让主角承受明显不公的压力
- 中段用多次忍耐积蓄观众的情绪
- 反击必须有策略感，不是蛮力
- 爽点之后必须有一层价值升华，避免沦为纯爽文

**名场面常出现在**：反击和逆转的高光时刻

---

## 5. 温情型

适用于：日常生活片、亲情广告、人文短片、治愈系

```
日常 → 微妙异常 → 误会 → 真相 → 情感击中 → 回响
```

**节奏要点**：
- 开头展示看似平淡的日常
- 一个微小的异常打破日常（习惯突然改变、熟悉的东西消失）
- 中段制造误会，让观众和角色都误解了真实原因
- 真相揭示时应该是一个安静的瞬间，不是激烈的冲突
- 用一个克制的动作完成情感击中

**名场面常出现在**：真相揭示后角色的一个沉默动作

---

## 如何选择情绪曲线

| 如果故事的核心是… | 推荐曲线 |
|------------------|----------|
| 人物关系和情感 | 情感型 / 温情型 |
| 信息差和认知翻转 | 悬念型 |
| 荒诞前提和社会讽刺 | 喜剧型 |
| 弱者逆袭和正义伸张 | 爽感型 |
| 日常生活中的隐藏感动 | 温情型 |
| 人性的复杂和灰色地带 | 情感型 / 悬念型 |

## 混合使用

高级剧本常常混合两种曲线：

- **悬念+情感**：用悬疑结构推进，用情感内核打动（例：真相是父亲一直在保护孩子）
- **喜剧+温情**：用荒诞开场，用温暖收尾（例：一个机器人学会了不完美的爱）
- **爽感+悬念**：用反击制造爽点，但真正的逆转在于信息差的揭示

混合时，主曲线决定整体节奏，副曲线负责情绪变化的丰富度。
